# Getting Started

### Prerequisites
This project runs on [Node.js](https://nodejs.org/en)
After installation, run
```
node -v
```
to check if node is installed correctly.

### Dev mode
run
```
npm run dev
```
and visit the server in [localhost:8080](http://localhost:8080)
